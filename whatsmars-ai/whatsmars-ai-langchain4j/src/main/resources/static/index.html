whatsmars-ai
基础 AI 对话接口，调用 GET /ai/chat?message=... 返回完整回复。
基于 TokenStream + SSE 实现流式输出，调用 GET /ai/stream/chat?message=...，实时显示 AI 逐字回复。
AI 根据问题自动调用工具函数，调用 GET /ai/function/chat?message=...，流式输出。
可用工具：获取当前时间/日期、加减乘除、天气查询、字符串操作等。
基于知识库的检索增强生成（RAG），调用 GET /ai/rag/chat?message=...，流式实时输出。
知识库包含 Spring Boot 核心知识和 Java 并发编程文档，AI 先检索相关内容再逐字回答。
基于 @MemoryId 的多轮对话记忆，流式输出，调用 GET /ai/memory/chat?sessionId=...&message=...。
相同 sessionId 的对话会保留上下文，不同 sessionId 互相隔离。
