Spring AI Demo

whatsmars-ai
基础 AI 对话，调用 GET /ai/chat?message=... 返回完整回复（非流式）。
流式 AI 对话，调用 GET /ai/chat/stream?message=...，实时逐字输出 AI 回复。
通过 System Message 设定 AI 角色为资深 Java 架构师，调用 POST /ai/advanced/system-message，流式输出。
Few-shot Prompting 提供示例引导 AI 将自然语言转换为 Java 代码，调用 POST /ai/advanced/few-shot，流式输出。
高温度创意性对话，AI 扮演作家角色，调用 POST /ai/advanced/creative，流式输出。
AI 自动调用工具函数，流式输出。可用工具：
天气查询 /ai/tool/weather、时间查询 /ai/tool/time、智能助手 /ai/tool/ask
ReAct Agent 智能问答，Agent 自动判断并调用工具，调用 GET /ai/react-agent/chat，流式输出。
基于知识库的 RAG 检索增强生成，调用 GET /ai/rag/query?question=...，流式输出。
基于 JDBC 持久化的多轮对话记忆，调用 GET /ai/memory/chat?conversationId=...&message=...，流式输出。
相同 conversationId 的对话共享上下文，不同 ID 互相隔离。
基于 PromptTemplate 的模板化调用，流式输出。支持产品描述、代码解释、自定义模板。
结构化输出，AI 返回 JSON 格式数据。调用 POST /ai/structured/extract-user 或 /review-summary。
带缓存的 AI 聊天，相同问题首次请求 AI API，后续直接返回缓存结果。调用 GET /ai/cache/chat。
多模态图像理解，调用 POST /ai/vision/analyze-url。输入图片 URL，AI 描述图片内容。
